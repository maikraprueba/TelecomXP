# Telecom X — Análisis de Churn

**Autor:** : Maycol Ninaja

## Descripción
Análisis compacto para identificar causas de abandono (churn) en Telecom X y proponer acciones de retención.

## Objetivos
- Identificar factores que influyen en el churn.
- Comparar clientes que se van vs. los que permanecen.
- Sugerir acciones para reducir la deserción.

## Archivos clave
- `ChallengeTelecomX.ipynb` — Notebook con el análisis.
- `TelecomX_Data.json` — Dataset (JSON, ~7k registros).
